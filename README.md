# 📚 KitabKart - eBook Store

## 🚀 Project Overview
This is a full-stack eBook store built using React, Node.js, Express, and MongoDB.

---

## 🛠️ Tech Stack
- Frontend: React (Vite)
- Backend: Node.js, Express
- Database: MongoDB
- Authentication: JWT
- Security: bcrypt, Role-Based Access

---

## 🔐 Features
- User Login & Signup
- JWT Authentication
- Password Hashing (bcrypt)
- Role-Based Access (Admin/User)
- Book Listing

---

## ⚙️ How to Run

### Backend
cd backend  
npm install  
npm run dev  

### Frontend
cd frontend  
npm install  
npm run dev
